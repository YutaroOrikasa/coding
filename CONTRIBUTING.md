### Case: When Using GitHub Copilot

When committing changes generated with the help of GitHub Copilot, clearly indicate that AI assistance was used. This ensures transparency in the collaboration process and helps others understand the source of the changes.

#### **Commit Message Format**
For commits authored with GitHub Copilot, include the following line at the end of the commit message:
```
(Created by GitHub Copilot)
```

#### **Example Commit Messages**
**1. Adding a new feature**
```
feat: Implement user authentication system

Added login and signup features, including JWT-based authentication.
(Created by GitHub Copilot)
```

**2. Fixing a bug**
```
fix: Resolve crash on submitting empty password

Caught and handled edge cases for empty password submissions during login.
(Created by GitHub Copilot)
```
